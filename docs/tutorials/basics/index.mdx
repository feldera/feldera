# Feldera Continuous Analytics Platform: the Basics

In this tutorial we cover the basics of using Feldera, including:

- Writing and testing SQL programs
- Building continuous analytics pipelines
- Managing pipelines using Feldera Web Console

Along the way, we will introduce fundamental concepts behind Feldera:

- Continuous analytics
- Incremental query evaluation
- Computation over data-in-motion

> **_NOTE:_**  Throughout the tutorial we will encounter limitations of the
> developer preview version of Feldera.  These limitations will be removed in the
> upcoming official release.
